# AI/ML Playground

Proyek ini adalah kerangka untuk membuat simulasi konsep AI/ML interaktif (mirip TensorFlow Playground) yang dapat di-deploy ke GitHub Pages.

## Struktur
- `index.html`
- `src/` inti aplikasi
- `styles/` style sheet
- `docs/` dokumentasi dan deployment
- `SKILL.md` rencana untuk implementasi step-by-step

## Mulai
1. `npm install`
2. `npm run dev`

## Deploy
Lihat `docs/deployment.md`.
